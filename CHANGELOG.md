# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-12

### Added
- Initial release
- Slack monitoring with 8-strategy search
- Bot/noise filtering with configurable bot list
- Priority scoring with configurable boosts and penalties
- Thread deduplication
- Claude Code CLI integration for AI-powered investigation
- Writer/Reviewer quality loop with 7 criteria
- Human approval flow via Slack interactive messages
- Learning engine with pattern tracking and feedback
- YAML + environment variable configuration
- macOS launchd daemon support
- Comprehensive test suite
